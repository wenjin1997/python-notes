# 练习
## 数字形式转换I
### 描述
获得用户输入的一个正整数输入，输出该数字对应的中文字符表示。  
0到9对应的中文字符分别是：零一二三四五六七八九  
### 输入输出示例
输入 123 输出 一二三  
输入 9876543210 输出 九八七六五四三二一零
### 代码
```
# 数字形式转换
template = "零一二三四五六七八九"
s = input()
for c in s:
    print(template[eval(c)], end="")
```
